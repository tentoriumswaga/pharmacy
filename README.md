## Описание

Аптечная информационная система. Реализует основные прецеденты:
1. **Просмотр препаратов** — поиск и фильтрация лекарств по названию, категории, производителю и наличию.
2. **Продажа** — оформление продажи препарата фармацевтом с изменением количества на складе.
3. **Редактирование** — поиск по ID и изменение данных о препарате (название, категория, производитель, количество, цена).
4. **Формирование отчета** — создание отчета по продажам за период с экспортом в TXT, CSV или JSON.

## Стек

- Python + Tkinter (GUI, входит в стандартную библиотеку)
- JSON (хранение данных)
- `unittest` + `coverage` (тестирование, покрытие логики ≥90%)
